# DesafioFullstackFront

## Instalação

```bash
$ npm install
```

## Inicialização

```bash
# development
$ ng serve
```
- Inicializar o projeto utilize o comando `ng serve`

## Conexão com a API

```bash
Acesse o arquivo `environment.ts`, e informe o IP da API no campo `baseUrl`.
```
