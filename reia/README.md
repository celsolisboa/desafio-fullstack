# Réia

Reio é o proejto de backend que alimenta o projeto PGC

## Setup & Run
1. acessar a pasta do projeto `reia`
2. rode `npm install`
3. rode `npm start`
4. Sistema estará rodando na url `http://127.0.0.1:3000/`

## Endpoints disponíveis

http://127.0.0.1:3000/explorer/#/
